Problem Statement

Our problem that we are trying to classify music based on 10 different genres.

Dataset

GTZAN Dataset, It contains 3 different formats:
1) WAV Files
2) Visual Representation Images like Spectogram, Mel Spectogram, and Waveforms
3) CSV file contains some music feature extracted from the song like the mean and the variance

It has been used by many papers which indicates how stable the dataset is It contains many formats that enable us to use many neural network architectures (CNN on the images) and (RNN for example on the wav files)

State of the art

1) CALM is a promising pre-training strategy for MIR. (2021)
2) CALM learns richer representations by modeling audio instead of labels.
3) It was used in this paper for different MIR tasks like our problem along with NLP Model on GTZAN dataset.
4) It achieved accuracy: 79.7%



Orignial Model from Literature

Our base model uses a CNN Model

The specifications of the model are as follows:
1) Input layer: 64 * 256 neurons, corresponds to 64 mel scales and 256 time windows(23ms, 50% overlap).
2) Convolution layer: 64 different 3 * 3 filters with a stride of 1.
3) Max pooling layer: 2 * 4.
4) Convolution layer: 64 different 3 * 5 filters with a stride of 1.
5) It achieved 70% Accuracy.
